# AI-Fitness-Trainer-YOLO
YOLOv8-Pose kullanarak gerçek zamanlı şınav sayan yapay zeka antrenörü.
Proje Amacı: Bu proje, sporcuların şınav formunu analiz eder ve otomatik olarak sayar.

Teknik Detay: YOLOv8-Pose mimarisi kullanılarak eklem noktaları (omuz, dirsek, bilek) takip edilir ve trigonometrik açı hesaplamalarıyla tekrar tespiti yapılır.

Nasıl Çalışır? python main.py komutuyla kamerayı başlatın ve yan profil olarak şınav çekmeye başlayın.
